## 📖 Overview

A static profile card UI built for the **Web Foundations** track, Day 1: HTML & CSS Fundamentals. The layout uses a two-panel design — a navy identity panel with a diagonal edge, and a content panel for bio, skills, and contact — built entirely with the CSS Box Model and Flexbox, with a single-column responsive fallback for mobile.

---

## ✨ Features

- 🖼️ Profile photo placeholder (SVG icon)
- 🧑 Name, profession & location
- 📝 About / bio section
- 🏷️ Skills displayed as tags
- 📩 Contact button (links out)
- 📱 Fully responsive — stacks vertically on screens ≤600px

## ⚙️ Installation & Setup

No build tools or dependencies required — it's a single static HTML file.

```bash
# Clone the repository
git clone https://github.com/atharva-9423/unprof.git

# Navigate to the Day 1 folder
cd unprof/day-1

# Open directly in your browser
open profile_card.html      # macOS
start profile_card.html      # Windows
```

Or just double-click `profile_card.html` — no server needed.

## 🧩 Concepts Applied

| Concept | Where it's used |
|---|---|
| **Semantic HTML** | `<article>`, `<section>`, `role="main"` |
| **CSS Box Model** | Padding, gap, and spacing across panels |
| **Flexbox** | Card layout, skill tags, button alignment |
| **CSS Variables** | Centralized color & radius tokens in `:root` |
| **clip-path** | Diagonal edge on the identity panel (signature element) |
| **Media Queries** | Mobile breakpoint at `max-width: 600px` |
