## ✨ Sections

| Section | Description |
|---|---|
| 🔝 Navbar | Sticky navigation bar with logo and smooth scroll links |
| 🦸 Hero | Full-width profile section with photo, stats, and CTA |
| 👤 About | Two-column layout with bio and highlights |
| 🛠️ Skills | Responsive 3-column skill cards grid |
| 📩 Contact | Dark CTA section with LinkedIn and GitHub links |
| 🦶 Footer | Simple branded footer |

---

## 🧠 CSS Concepts Demonstrated

| Concept | Where It's Used |
|---|---|
| **Flexbox** | Navbar, Hero layout, Stats row, Contact buttons |
| **CSS Grid** | About section (2 columns), Skills section (3 columns) |
| **Media Queries** | Breakpoints at 768px (tablet) and 600px (mobile) |
| **CSS Variables** | Full color palette defined in `:root` |
| **clip-path** | Diagonal cut at the bottom of the Hero section |
| **Semantic HTML** | `<nav>`, `<header>`, `<section>`, `<footer>`, `aria-label` |

---

## 📱 Responsive Behavior

| Screen Size | Layout |
|---|---|
| Desktop (900px+) | Full side-by-side layouts, 3-col skills grid |
| Tablet (768px) | 2-col skills grid, stacked About section |
| Mobile (600px) | Everything stacked, simplified navbar |

## 🚀 How To Run Locally

No installation needed.

**Step 1** — Clone the repository:
```bash
git clone https://github.com/atharva-9423/unprof.git
cd day-2
```

**Step 2** — Open in your browser:
```bash
# Windows
start profile-page.html

# Mac
open profile-page.html
```

Or just double-click `profile-page.html` — no server needed!
